我为什么要搭建一个博客

  • iosTangtang
  • 2 Minutes
  • 2017年5月17日

起因

怎么说呢，很早之前就想过要自己搭一个博客，甚至说包括博客的前端和后台都是自己写的，但是很遗憾，自己的能力还是有限，没有办法在课余时间将前端和后台都学完，现在想想还是挺遗憾的。萌生搭博客的想法大概是在大二上学期的时候吧，那时候也有些年轻气盛，什么都想尝试着去做做，然而这个想法在萌生之后，却一直没有去实现。之后的时间都花在了 iOS 开发上了，写项目，学知识啥的，这个想法就一直被搁置了。

为什么最近实现了这个想法呢？大概还是因为，有点闲吧。在找到实习之后的时间里，无法再继续学习下去，整个懒得不行，然后，有一天实在是无聊的不行了，又看见实验室以前的一个学长开通了自己的公众号，忽然意识到自己连个博客都没有，所以就上手开始搭一个自己的博客。

个人简介

1. 学历

目前是一枚大三狗，过了这个暑假之后就荣升大四狗了。就读在西安邮电大学软件工程专业，当初高考的时候考的分数很尴尬，不高不低，去哪儿都不知道，最后我也不知道为啥就来到了西邮。但感觉也算误打误撞来到了个不错的学校。

2. 学习方向

目前是一枚 iOS Coder，也不能说学的特别好，能写写一般的项目，大型的项目没试过，最近也有想法去试试。除此之外呢，学校的课还开了 Android、前端和 Java Web，写过一段时间的 Java Web，最后也因为要找实习就没再去学了。

3. GItHub 脑残粉

使用 GitHub 大概是在大二下学期的时候，那时候是在实验室学长（西邮移动应用开发实验室）的推荐下看了一下，感觉挺好用的，当然一开始太年轻，只是作为一个代码托管的仓库，作用大概有点像某云。现在的话会上去浏览一些开源项目啥的，看看别人开源的代码格式，当然绝大部分是 iOS 方面的，慢慢就陷入了这个全球最大的同性交友平台。

4.扯淡

我也不知道上面写了那么多都在写什么，其实之后的打算还是蛮多的，不然也不会选择自己搭建一个博客同时注册了一个域名。希望能激励一下自己。

搭建博客的经过

在不会写前端和后台的前提下，果断选择了 GitHub+Hexo 来搭建个人博客，毕竟 GitHub 脑残粉。整个过程其实也不长，后面会专门写一篇博文来说明，这里就先不说了。中间出过几个岔子，包括忘了开 root 权限、漏掉了部分插件、域名更换出现问题等等。当然都不是很大的问题，最后还是成功搭上了，挺 Happy 的。花的时间也不多，主要是踩了几个坑，大概整个过程大概花了两个多小时左右。域名注册的时候第一次本来想用 Tangtang.com，但是被注册了，最后换成了 iostangtang.com。注册域名的钱，大概就是我写博客的激励吧，233333。

最后

我也不知道我到底写了个啥，反正就是搭了个博客，然后看博客上空空的，随便扯扯淡作为第一篇博客。以后的话呢，想每周定期更新一篇博客，虽然我也不知道我这个懒鬼会不会去更，反正先立个小 flag：

每周更新一篇博客！！！

我以后尽量做到，做不到想想域名的钱，233333。然后如果有人能看到我的博客（好心酸的感觉），希望能一起相互交流学习。
